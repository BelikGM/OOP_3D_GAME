Об игре:
Игра была разработана на движке Unity 2021.3.6f1 на языке C#
Запуск:
Для запуска игры нужно открыть папку Игра, затем папку SphereX, запустить файл My Project.exe
Системные требования:
OC: Microsoft Windows 10
Mac OS 10.6+
Процессор: с тактовой частотой 1.2 ГГц
Оперативная память: 512 МБ
Видеокарта: с 32 МБ видеопамяти, совместимая с Direct
Жесткий диск: 100 МБ свободного места
Управление:
A - Влево
D - Вправо
Space - Прыжок
ЛКМ - Выбор кнопок меню
Escape - Включить/Выключить меню паузы
